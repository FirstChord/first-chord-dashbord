Export Format: Creates a CSV file compatible with Anki import. Reverse cards will create two entries (front→back and back→front).
Back to Dashboard
How to Import
  1. Open Anki on your computer
  2. Go to File → Import
  3. Select the exported CSV file
  4. Choose your note type and deck
  5. Map the columns correctly
  6. Click Import
CSV Format
Columns:
  • Front - Question/Prompt
  • Back - Answer/Response
  • Type - Card type
  • Deck - Target deck
  • Tags - Space-separated tags
  • Extra - Additional notes
  • Guid - Unique identifier
